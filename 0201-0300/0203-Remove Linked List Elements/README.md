# 203. Remove Linked List Elements

Difficulty: Easy

https://leetcode.com/problems/remove-linked-list-elements/description/

Remove all elements from a linked list of integers that have value val.

**Example**  
**Given:** 1 --> 2 --> 6 --> 3 --> 4 --> 5 --> 6, val = 6  
**eturn:** 1 --> 2 --> 3 --> 4 --> 5
